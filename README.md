# Vehicle Service Management System

This is a simple Java project developed to demonstrate Object-Oriented Programming (OOP) concepts.

## Features
- Abstract class (VehicleService)
- Inheritance
- Method Overriding
- Service charge calculation for Cars and Bikes
- Simple console-based application

## Technologies Used
- Java
- Eclipse/VS Code

## Concepts Covered
- Classes and Objects
- Abstraction
- Inheritance
- Abstract Methods
- Method Overriding
- Polymorphism

## Output
The program accepts vehicle service details and calculates the service charge based on the vehicle type.
